# Política de Segurança

## Versões Suportadas

Apenas a versão mais recente da branch `main` recebe atualizações de segurança.

| Versão | Suportada          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reportando uma Vulnerabilidade

Se você descobrir uma falha de segurança neste projeto, por favor, **NÃO abra uma Issue pública**.

Em vez disso, envie um e-mail para **doblerveni@gmail.com**  com os detalhes.
Nós responderemos em até 48 horas.
