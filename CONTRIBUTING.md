# Contribuindo para o EQFAR LIMS

Obrigado por investir seu tempo em contribuir para o nosso projeto!

## Como posso ajudar?

### Reportando Bugs
Se você encontrar um bug, abra uma **Issue** detalhando:
- O que você estava tentando fazer.
- O que aconteceu.
- O que você esperava que acontecesse.
- Passos para reproduzir o erro.

### Sugerindo Melhorias
Abra uma **Issue** com a tag `enhancement` descrevendo sua ideia.

### Pull Requests
1. Garanta que seu código segue o padrão PEP-8 (Python).
2. Use commits semânticos (ex: `feat:`, `fix:`, `docs:`).
3. Atualize a documentação se necessário.
